# FamilyBudget AI Frontend

React + TypeScript + Vite frontend for FamilyBudget AI.

## Requirements

- Node.js 22+
- Backend running at `http://localhost:5001`

## Environment

`.env`:

```text
VITE_API_BASE_URL=http://localhost:5001
```

## Run Frontend

```bash
npm install
npm run dev
```

App URL:

```text
http://localhost:5173
```

## Pages

- `/`
- `/login`
- `/register`
- `/dashboard`
- `/subscriptions`
- `/assistant`
- `/assistant/chat/:id`
